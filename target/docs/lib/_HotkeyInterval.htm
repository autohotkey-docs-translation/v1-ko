#HotkeyInterval

#MaxHotkeysPerInterval과 함께, 핫키의 활성화 비율을 지정합니다. 이 값을 넘어서면 경고 대화상자가 타나납니다.

#HotkeyInterval Milliseconds

매개변수

Milliseconds

밀리초 단위 간격.

논평

스크립트에 이 지시어를 지정하지 않으면 마치 2000을 지정한 것처럼 행위합니다.

자세한 내용과 논평은 #MaxHotkeysPerInterval를 참조하십시오.

다른 지시어처럼, #HotkeyInterval는 조건적으로 실행할 수 없습니다.

관련 항목

#MaxHotkeysPerInterval

예제

Allows a maximum of 200 hotkeys to be pressed within 2000 ms without triggering a warning dialog.

#HotkeyInterval 2000  ; 이것이 기본값입니다 (밀리초).
#MaxHotkeysPerInterval 200
