EnvMult

주어진 값을 변수에 곱해 그 변수에 다시 설정합니다. 다음과 동등합니다: Var *= Value.

비추천: 이 명령어는 새 스크립트에 사용을 추천하지 않습니다. 대신 Var := Var * Value 또는 Var *= Value를 사용하십시오.

EnvMult, Var, Value

매개변수

Var

연산을 할 변수의 이름.

Value

정수, 부동 소수점 수, 또는 표현식.

논평

이 명령어는 다음 단축 스타일과 동등합니다: Var *= Value.

Var나 Value가 비어 있거나 또는 숫자로 시작하지 않으면, 계산을 위해 0으로 간주됩니다 (단 내부적으로 표현식에 Var := X *= Y와 같이 사용할 경우는 제외합니다).

Var나 Value에 소수점이 들어 있다면, 결과는 부동 소수점 수가 되고 형식은 SetFormat에 의해 설정됩니다.

관련 항목

EnvAdd, EnvSub, EnvDiv, SetFormat, 표현식, If var is [not] type, SetEnv, 비트별 연산 (변형)

예제

Sets MyCount to itself times 2.

EnvMult, MyCount, 2

위와 동등.

MyCount *= 2
