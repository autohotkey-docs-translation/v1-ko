#KeyHistory

KeyHistory 창에 보여지는 키보드와 마우스 이벤트의 최대 개수를 설정합니다. 0을 설정하면 키 이력이 꺼집니다.

#KeyHistory MaxEvents

매개변수

MaxEvents

KeyHistory 창에 보여지는 키보드와 마우스 이벤트의 최대 개수 (기본값은 40이고, 최대는 500). 0을 지정하면 키 이력이 완전히 꺼집니다.

논평

이 설정은 스크립트가 실행을 시작하기 전에 효과를 내기 때문에, (스크립트 어느 곳에서나) 한 번만 지정하면 됩니다.

각 키눌림과 마우스 클릭은 다운-이벤트와 업-이벤트로 구성되기 때문에, KeyHistory는 "완전한 이벤트"의 반쪽만 지정된 갯수만큼 보여줍니다. 예를 들어, 스크립트에 #KeyHistory 50이 들어 있다면, 키눌림과 마우스 클릭을 25 개까지 보여줍니다

다른 지시어처럼, #KeyHistory는 조건적으로 실행할 수 없습니다.

관련 항목

KeyHistory, #NoTrayIcon

예제

Causes KeyHistory to display the last 100 instead 40 keyboard and mouse events.

#KeyHistory 100

Disables key history entirely.

#KeyHistory 0
