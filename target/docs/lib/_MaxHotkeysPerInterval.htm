#MaxHotkeysPerInterval

#HotkeyInterval과 함께, 핫키 활성화 비율을 지정합니다. 이 값을 넘어서면 경고 대화상자가 나타납니다.

#MaxHotkeysPerInterval Value

매개변수

Value

경고 대화상자를 촉발하지 않고 #HotkeyInterval가 지정한 시간 사이에 누를 수 있는 핫키의 최대 개수.

논평

너무 관대하게 지정하지 않도록 주의하십시오. 왜냐하면 의도치 않게 무한 키눌림 회돌이를 초래한다면 (Send 명령어를 사용했는데 우연하게도 다른 핫키를 촉발하게 되면), 범람하는 키보드 이벤트 때문에 컴퓨터가 응답을 하지 못하게 될 수 있습니다.

좀 과도하게 간략한 예제를 들자면, 핫키 ^c::Send ^c는 무한 키눌림 회돌이를 생산합니다. 이를 피하려면, 핫키 정의에 $ prefix 접두사를 추가해 (예, $^c::) 핫키가 Send 명령어로 촉발되지 못하게 하십시오.

이 지시어가 스크립트 안에 지정되어 있지 않으면, 70으로 설정한 것처럼 행위합니다.

다른 지시어처럼, #MaxHotkeysPerInterval는 조건적으로 실행할 수 없습니다.

예제

Allows to press a maximum of 200 instead of 70 hotkeys per interval.

#MaxHotkeysPerInterval 200
