EnvSet

값을 환경 변수에 씁니다.

EnvSet, EnvVar, Value

매개변수

EnvVar
사용할 환경 변수의 이름. 예, "COMSPEC" 또는 "PATH".
Value

환경 변수에 설정할 값.

에러 처리

[v1.1.04+]: 이 명령어는 실패하면 예외를 던질 수 있습니다. 더 자세한 정보는 실행시간 에러를 참조하십시오.

ErrorLevel은 문제가 있으면 1, 그렇지 않으면 0이 설정됩니다.

논평

운영 체제는 환경 변수의 길이를 32 KB의 텍스트로 제한합니다.

이 명령어로 생성되거나 변경된 환경 변수는 스크립트가 Run 또는 RunWait를 통하여 실행한 프로그램에서만 접근이 가능합니다. 더 자세한 것은 환경 변수를 참조하십시오.

이 명령어는 SetEnv와 별도로 존재합니다. 왜냐하면 정상적인 스크립트 변수는 운영 체제의 환경에 저장되지 않기 때문입니다. 수행성능이 악화되고 OS가 환경 변수를 32 KB의 길이로 제한하기 때문입니다.

관련 항목

EnvGet, #NoEnv, 환경 변수, EnvUpdate, SetEnv, Run, RunWait

예제

Writes some text to the AutGUI variable contained in the environment.

EnvSet, AutGUI, 변수에 넣을 텍스트.
