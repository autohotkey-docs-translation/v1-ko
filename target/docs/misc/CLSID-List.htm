CLSID List (Windows Class Identifiers)

운영체제 안의 어떤 특수 폴더는 유일한 문자열로 식별됩니다. 이 문자열은 FileSelectFile, FileSelectFolder, 그리고 Run에 사용할 수 있습니다. 예를 들어:

FileSelectFile, OutputVar,, ::{645ff040-5081-101b-9f08-00aa002f954e}  ; 휴지통 안에서 파일 하나를 선택합니다.
FileSelectFolder, OutputVar, ::{20d04fe0-3aea-1069-a2d8-08002b30309d}  ; 내 컴퓨터 안에서 폴더를 하나 선택합니다.

마지막 컬럼에 있는 "Yes" 엔트리들은 절대적이 아닙니다: Run 명령어는 시스템 환경 구성에 따라 다른 CLSID를 지원할 수도 있습니다. Run을 통하여 CLSID를 열려면, 그냥 첫 매개변수로 CLSID를 지정하면 됩니다. 예를 들어:

Run ::{20d04fe0-3aea-1069-a2d8-08002b30309d}  ; "내 컴퓨터" 폴더를 엽니다.
Run ::{645ff040-5081-101b-9f08-00aa002f954e}  ; 휴지통을 엽니다.
Run ::{450d8fba-ad25-11d0-98a8-0800361b1103}\My Folder  ; "내 문서" 안에 존재하는 폴더를 엽니다.
Run %A_MyDocuments%\My Folder  ; 대부분의 시스템에서 위와 같습니다.
