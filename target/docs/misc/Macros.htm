키보드 매크로나 마우스 매크로 만들기

매크로는 요구하면 "플레이되는" 스크립트된 일련의 조치입니다; 가장 흔한 매크로 조치는 하나 이상의 창에 흉내낸 키눌림 그리고 마우스 클릭을 전송하는 것입니다. 그런 창은 각 키눌림과 마우스 클릭에 마치 여러분이 손수 수행한 것처럼 반응합니다. 그러므로 반복 작업을 빠르게 그리고 신뢰성 있게 자동화할 수 있습니다.

매크로는 손으로 작성할 수 있지만, 기다란 매크로를 작성하려면 매크로 레코더의 도움을 받은 편이 쉬울 것입니다. 예를 들어 Titan/polyethene가 작성한 레코더 스크립트가 있습니다. 여러분이 타자하고 클릭하는 것을 지켜 보면서 활성 창을 추적 관리합니다. 이 조치를 작동하는 매크로로 변환하면 나중에 더 빠른 속도로 "다시 플레이 할" 수 있습니다.

매크로를 다시 플레이하는 가장 편리한 방법중 하나는 그것을 핫키나 핫스트링에 할당하는 것입니다. 예를 들어, 다음 핫키는 빈 이메일 메시지를 생성하고 그것을 특정 유형의 수신인을 위해 준비합니다. 그것을 보내기 전에 개별적으로 바꿀 수 있습니다:

^!s::  ; Control+Alt+S 핫키.
IfWinNotExist Inbox - Microsoft Outlook
    return  ; Outlook이 오른쪽에 열리지 않았습니다. 그래서 아무것도 하지 않습니다.
WinActivate  ; 위의 명령어로 발견한 창을 활성화합니다.
Send ^n  ; Control+N을 통하여 새로 빈 이메일을 생성합니다.
WinWaitActive Untitled Message
Send {Tab 2}Product Recall for ACME Rocket Skates  ; 제목 줄을 설정합니다.
Send {Tab}Dear Sir or Madam,{Enter 2}We have recently discovered a minor defect ...  ; 등등.
return  ; 이 줄은 핫키를 종료하는 데 기여합니다.

위와 같은 핫키 매크로는 매일 자주 수행해야 하는 작업에 특히 유용합니다. 대조적으로, 자주 사용하지 않는 매크로는 각각 별도의 파일에 보관하고 바탕화면이나 시작 메뉴의 단축 아이콘을 통하여 접근합니다.
